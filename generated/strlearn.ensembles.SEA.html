strlearn.ensembles.SEA¶

class strlearn.ensembles.SEA(base_estimator=None, n_estimators=10, metric=<function accuracy_score>)¶

Streaming Ensemble Algorithm.

Ensemble classifier composed of estimators trained on the fixed number of previously seen data chunks, prunning the worst one in the pool.

Parameters
n_estimatorsinteger, optional (default=10)

The maximum number of estimators trained using consecutive data chunks and maintained in the ensemble.

metricfunction, optional (default=accuracy_score)

The metric used to prune the worst classifier in the pool.

Examples

>>> import strlearn as sl
>>> stream = sl.streams.StreamGenerator()
>>> clf = sl.ensembles.SEA()
>>> evaluator = sl.evaluators.TestThenTrainEvaluator()
>>> evaluator.process(clf, stream)
>>> print(evaluator.scores_)
